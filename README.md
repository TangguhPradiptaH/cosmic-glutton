# Cosmic Glutton

Cosmic Glutton adalah gim web sederhana berbasis browser yang dikembangkan sebagai proyek pembelajaran dan portofolio. Gim ini mengusung konsep arcade dengan sistem leaderboard yang menyimpan sepuluh skor tertinggi secara otomatis menggunakan API berbasis PHP dan database MySQL.

Proyek ini dirancang untuk menunjukkan pemahaman dasar mengenai pengembangan web, pengelolaan basis data, serta penerapan prinsip CRUD pada sistem nyata.

---

## Teknologi yang Digunakan

* HTML untuk struktur antarmuka
* CSS dan Tailwind CSS untuk tampilan dan tata letak
* JavaScript untuk logika permainan dan komunikasi API
* PHP untuk backend dan pengolahan data
* MySQL untuk penyimpanan leaderboard

---

## Fitur Utama

* Gameplay berbasis browser yang ringan dan responsif
* Sistem leaderboard otomatis sepuluh besar
* Penyimpanan nama dan skor pemain setelah permainan berakhir
* Penghapusan otomatis skor di luar sepuluh besar
* Antarmuka sederhana dengan nuansa galactic

---

## Struktur Proyek

* index.html sebagai halaman utama permainan
* api.php sebagai endpoint API leaderboard
* config.php sebagai konfigurasi koneksi database
* assets sebagai folder pendukung seperti JavaScript dan aset visual

---

## Cara Menjalankan Proyek

1. Clone repository ini ke dalam folder htdocs
2. Jalankan XAMPP atau server lokal sejenis
3. Import database MySQL yang telah disediakan
4. Pastikan konfigurasi database sesuai pada file config.php
5. Akses proyek melalui browser menggunakan localhost

---
